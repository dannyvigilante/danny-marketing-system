## QA Voice Rubric

When QA Reviewer scores voice compliance, it checks:

| Dimension | What it means for this brand |
|---|---|
| Brand voice match | Does it sound like [Brand]? |
| Audience resonance | Would [target reader] feel seen? |
| Key message presence | Does it reinforce a brand pillar? |
| Non-negotiable check | Does it avoid hard-line language/topics? |
| Format fit | Is it right for the channel it's written for? |

Minimum passing score: **4.0 / 5.0** across all dimensions.
